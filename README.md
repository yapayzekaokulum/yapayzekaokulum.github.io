# yapayzekaokulum.github.io
Yapayzekaokulum, yapay zeka ve yazılım alanlarında kurumsal eğitim ve danışmanlık sunarak işletmelerin kurumsal dönüşümüne rehberlik eder. Amacımız, teknik konuları iş dünyası için anlaşılır ve uygulanabilir kılmaktır. Girişimci vizyonunu Dr. Murat Altun’un danışmanlığıyla birleştirerek, iş ortaklarımızın hedeflerine ulaşmasına yardımcı oluyoruz.
